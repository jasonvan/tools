About This Tool

Convert PDFs between Letter, Legal, Tabloid, A3, A4, and A5 sizes

How It Works

This tool resizes PDF pages while preserving aspect ratio. Content is scaled to fit the new page size and centered automatically.

Limitations

  • Form fillable PDFs: form fields may get adjusted or misaligned when the PDF is scaled
  • Encrypted/password-protected PDFs are not supported
  • Maximum file size is 50MB
  • Mixed page sizes will be converted uniformly
  • Output files may be larger than originals (no compression)

Browser Compatibility: Works best in modern browsers (Chrome, Firefox, Safari, Edge). All processing happens in your browser - no files are uploaded to any server.
